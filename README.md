# multi-tap-cli

Converts words into a string of numbers that represent the buttons that would be pressed on multi tap keypad common on older pre-touchscreen devices.

    |  @_@  |  abc  |  def  |
    |   1   |   2   |   3   |
    |  ghi  |  jlk  |  mno  |
    |   4   |   5   |   6   |
    |  pqrs |  tuv  |  wxyz |
    |   7   |   8   |   9   |
    |   *   |   0   |   #   |
    
    HELLO WORLD => 43556 96753
